# Video-Juego

Este proyecto es un juego interactivo en linea, que permite jugar con otro participante, desarrollado con HTML, CSS Y JS.

## Tecnologías usadas
- HTML
- CSS
- JS

## Características
- Diseño, Dinámica, interactiva, on-line.
- Diseño responsivo.
